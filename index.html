Simon Says Game

Press any key to start a Game

Game Instructions

1. Press any key to start the game.

2. Follow the sequence of colors shown by the game.

3. Click on the color boxes in the same order to Level Up!

4. If you make a mistake, the game will reset.

5. Each level will increase the sequence length, you have to enter the previous level sequence in correct order.

6. You have 2 seconds to enter the sequence, if you fail, the game will reset.
